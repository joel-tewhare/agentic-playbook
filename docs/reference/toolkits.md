## Toolkits

Short notes on higher-level libraries/SDKs and what problems they solve.

### Vercel AI SDK

- Purpose: Simplifies working with AI models (streaming, message formatting, multi-provider support)
- Replaces: manual API calls + streaming logic
- Mental model: App → SDK → Model

### React Query

- Purpose: Manages server data fetching, caching, and syncing
- Replaces: manual fetch + loading state + caching
- Mental model: UI subscribes to server state

### shadcn/ui

- Purpose: Prebuilt UI components
- Replaces: building UI primitives from scratch
- Mental model: styled building blocks
