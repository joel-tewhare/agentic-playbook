# Agentic Playbook

A personal playbook for using AI-assisted development in a structured way.

This repo documents the workflows, planning templates, review loops, and reusable skills I use to plan, build, review, and refine software projects with AI support.

## What this shows

- A four-pass build workflow: UI/Layout, Data Wiring, Derived Logic, Final Polish  
- Planning templates for turning ideas into scoped features  
- Review and retro patterns for improving code and process  
- Security-aware thinking around AI-assisted development  

## Why I made this

I created this while learning agentic coding workflows after Dev Academy, to make my use of AI more deliberate, reviewable, and focused on building better software rather than just generating code.
